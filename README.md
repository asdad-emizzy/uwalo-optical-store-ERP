# Optical ERP

Enterprise Resource Planning system for optical retail businesses.

## Features

- Patient Management
- Order Processing
- Lab Integration
- Inventory Management
- Prescription Handling

## Tech Stack

- Backend: NestJS/TypeScript
- Database: PostgreSQL
- Cache: Redis
- Message Queue: RabbitMQ

## Getting Started

```bash
npm install
npm run start:dev
```
